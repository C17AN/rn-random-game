'리액트 네이티브는 성능이 별로야!', '리액트 네이티브를 어느 대기업에서 써?!' 라는 기존의 편견과는 다르게, 친숙한 리액트로 모바일 어플리케이션을 제작할 수 있다는 것 자체로도 충분히 매력을 느낄 수 있었습니다.

취업을 바라보면서 무언가를 배우는 것이 아닌, 순수한 호기심으로 무언가를 배운다는 것이 얼마나 즐거운지 느낄 수 있었던 시간이었습니다 :)
